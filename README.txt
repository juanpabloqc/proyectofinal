-----------------Detector de instrumento----------------------

Se debe tener instaladas las sigueintes librerias:
- os          
- numpy       
- pickle      
- scipy       
- sys        
- sklearn
- tkinter
- simpleaudio

Para correr el programa solo es necesario ejecutar
interfaz.py, si se presenta algun error, se recomienda 
volver a correr clasificacion.py, para que el archivo .sav 
donde se encuentra guardado el clasificador se actualize en 
su ordenador y de nuevo volver a correr interfaz.py.

En la carpeta ya se encuentran los archivos .txt con las 
caracteristicas de todos los audios que se utilizaron para 
entrenar el clasificasdor, igualmente se encuentran los 
audios de prueba para ejecutar el programa, tambien se 
encuentran los archivos .py donde estan todas las funciones 
que se utilizan.

Si se desea extraer nuevamente las caracteristicas, se debe 
descargar la base de datos de los audios del link que se 
enecuentra al final.
